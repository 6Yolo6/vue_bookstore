# vue_bookstore 网上书城

> 本项目前后端分离，使用vue2+springboot+sqlite+jdk1.8

> 用到了vant组件以及elementUi，mybatis-plus

> 后端代码采用四层架构编写

## 源码

code目录下分为`back`和`front`前后端代码

## 文档

document目录下为项目文档
